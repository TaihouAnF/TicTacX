# Description / 简介
A Simple Tic Tac Toe supporting both PVP(local) and PVE, built by Unity3D.

一款使用Unity3D制作的简单井字棋，可支持PVP(本地)与PVE模式。

P.S. **分辨率目前仅为800*600**

## Feature / 功能
1. Basic Tic tac toe / 基本井字棋
2. VS AI / 对战AI
3. Scoring / 计分
4. Restart / 重开
